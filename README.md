# Movie Recommendation System

This project is a simple Movie Recommendation System built using Python and Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Recommends movies based on genre similarity
- Uses cosine similarity algorithm
- Beginner-friendly machine learning project

## How to Run
1. Install required libraries
2. Run:
python movie_recommendation.py

## Sample Output
Movies recommended for Inception:

- Interstellar
- Gravity
- Doctor Strange
